Experiment classes#

class aequitas.flow.experiment.Experiment(config_file: Path | None = None, config: DictConfig | dict | None = None, default_fields: Iterable[str] = ('methods', 'datasets'), save_artifacts: bool = True, save_folder: Path | None = PosixPath('artifacts'), artifacts: Iterable[str] = ('results', 'methods', 'predictions'), name: str | None = None)[source]#

Class to run a FairML Experiment.

This class allows to test multiple methods on multiple or a single dataset, with configurable hyperparameter optimization algorithms.

Parameters:

  • config_file (Union[pathlib.Path, str], optional) – Path to the experiment’s configuration file.

  • config (Union[DictConfig, dict], optional) – Configuration object to run the experiment.

  • default_fields (Iterable[str], optional) – Default fields to include in the configuration, by default (“methods”, “datasets”).

  • save_artifacts (bool, default True) – Whether to save artifacts (i.e., results of the experiment) or not.

  • artifacts (Iterable[str], default ("results", "methods", "predictions")) –

    Artifacts to save. These can be:

    • ”results”: The result files with metrics.

    • ”methods”: The models created by the experiment (pickle format).

    • ”predictions”: The predictions made by the models (parquet format). This saves both the scores and binarized predictions.

  • save_folder (pathlib.Path, default "./artifacts") – Path to the folder where artifacts will be saved.

  • name (str, optional) – Name of the experiment, by default None. If None, the name will be generated as an unique hash of the configuration file.

Example

>>> from aequitas.flow.experiment import Experiment
>>> experiment = Experiment(config_file="configs/default.yaml")
>>> experiment.run()
