Aequitas Flow API
=================

Welcome to the Aequitas Flow API documentation. 
This API provides a comprehensive suite of tools designed to facilitate experiments, manage datasets, and apply various methods for training fair models. 

Experiments Module
------------------

The Experiments module offers functionalities for setting up, running, and managing experimental workflows. 
It supports the integration of various datasets and methods to allow for flexible and robust experimental designs.
