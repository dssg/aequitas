30 seconds to the webapp¶

You can run the same webapp available online locally.

After installing Aequitas, run the following command in your terminal.

python aequitas_webapp.basic_upload

* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

Load data¶

Use the browse ... button to select a properly formated csv. (see input data) Hit upload.

Customize report¶

  • choose attributes
  • choose reference group by attribute
  • choose metrics

Get Fairness Report¶

The webapp returns descriptive interpretation of the results along with three sets of tables. * Fairness Measures Results * Bias Metrics Results * Group Metrics Results
